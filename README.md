## 一、项目介绍
本项目采用前后端分离模式，使用 SpringBoot、Vue、Mysql、Mybatis 等技术开发。主要包括
系统前台（企业简介、职位查询、简历投递）、系统后台（人事管理、薪资管理、招聘管理、统计分析）等模块。
